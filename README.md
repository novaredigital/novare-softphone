# Nováre Phone

Nováre Telecom's softphone app (iOS first, Android second). Native Swift + Kotlin on the Linphone SDK.

- The app ships with NO server addresses, ports, URLs, or credentials. Everything arrives in the QR payload at sign-in.
- Works against any Nováre PBX server, current or future, on any port.
- Push wake: PBX push gateway -> APNs VoIP push -> CallKit.
